# Mine Imator Simply Remade: Nuxi

For all your Minecraft animation needs!

## Dev environment

Requirements

- Dotnet 9.0+
- The Monogame plugin for your IDE of choice

## Setup

1. Clone the repo
2. Run `dotnet restore`
3. Run `dotnet build`
4. Run `dotnet run`
